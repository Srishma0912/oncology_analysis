# oncology_analysis

## 📊 Oncology Treatment Outcomes Analysis

### 📌 Overview

This project analyzes oncology patient data to evaluate survival rates, treatment costs, and hospital performance.

---

### ❓ Key Questions

* Which hospital has the highest survival rate?
* Does higher cost lead to better outcomes?
* How does cancer stage impact survival?

---

### 🛠 Tools Used

* SQL

---

### 📈 Key Insights

* Mayo Clinic shows the highest survival rate (~63.7%)
* Survival rates are similar across hospitals (~60–64%)
* Higher costs do not always result in better outcomes
* Early-stage cancer has significantly higher survival rates

---

### ⚠️ Limitations

* No control for patient demographics
* Results are correlational, not causal

---

### 📂 Files

* oncology_analysis.sql → SQL queries used for analysis
